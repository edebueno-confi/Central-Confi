Como cadastrar Lojas Físicas

Para cadastrar novas lojas físicas no seu e-commerce, siga os passos abaixo:

  1. Acesse a área administrativa da plataforma.

  2. No menu, clique em Configurações > Lojas Físicas.

Nesta página, você poderá visualizar e editar todas as lojas físicas cadastradas, seja como ponto de devolução para os consumidores ou como endereço de retorno para devoluções postadas ou coletadas.

Como cadastrar uma nova loja:

  1. Clique no botão "+" (conforme mostrado na imagem abaixo).

  2. Você será direcionado para a página Cadastro de Loja Física, onde deverá adicionar os dados da loja.

  3. Após preencher os dados, marque a opção “ATIVO” para ativar a loja no sistema.


Definindo o tipo de loja:

  • Parceiro Logístico: Se a loja for um endereço de retorno para devoluções (retorno para o seller), marque essa opção.

  • Entrega Física: Se a loja for um ponto de devolução para o consumidor (onde ele poderá devolver a reversa), selecione esta opção.

Após preencher todos os dados e selecionar o tipo de loja, basta salvar as alterações no final da tela.
