Como alterar ou aprovar os produtos de uma solicitação?

O e-commerce pode alterar os produtos ou aprovar todos os itens adicionados pelo consumidor na solicitação. Para isso, siga os passos abaixo:

  1. Acesse o menu "Solicitações".

  2. Abra a solicitação que você deseja alterar.

  3. Na aba "Ações Pendentes", você encontrará os botões "Aprovar Todos" ou "Alterar Produto(s)", conforme ilustrado na imagem abaixo.
