COMO CRIAR UM USUÁRIO
 Neste guia, mostramos como criar e configurar um novo usuário no painel.
 Acessando o painel:

 Ao lado esquerdo da loja selecionada, clique no botão “+” (conforme imagem abaixo).

 Cadastro de Usuários:

 Após selecionar “USUÁRIOS”, você será direcionado ao painel de Cadastro de Usuários , onde deverá incluir os dados abaixo:

 Nome (Orientamos a sempre colocar o nome do usuario + setor ou cargo)
 E-mail (Login)
 Senha
 Cliente:
 Tipo:

 Além disso, há configurações adicionais que você pode ajustar:
 Autenticação de dois fatores : Ao ativar essa opção, o usuário receberá um código de verificação por e-mail após inserir o login (e-mail e senha), sendo necessário informar o código para concluir o acesso à área administrativa.

 Filtro de status de processo : Defina os status de reversas que o usuário poderá visualizar.

 Filtro de pendências : Configure os filtros de pendências que o usuário poderá acessar.

 Sellers : Onde deve ser adicionado os sellers que esse usuario terá acesso

 Habilitar notificação de nova solicitação por seller: Onde os usuários serão notificados via e-mail caso a solicitação pertença a um dos sellers que recebeu permissão de acesso.

 Finalizando:

 Após preencher as informações e ajustar as configurações, clique no botão “Salvar” para concluir a criação do usuário.
