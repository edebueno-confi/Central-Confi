Posso alterar o status de uma solicitação?

Sim, o e-commerce pode alterar manualmente o status de uma solicitação. Para isso, siga os passos abaixo:

  1. Acesse a área administrativa da plataforma.

  2. No menu lateral, clique em "Solicitações".

  3. Abra a solicitação cujo status você deseja alterar.

  4. Na aba "Resumo da Solicitação", localize a seção "Dados Gerais".

  5. O botão para alteração de status estará disponível nesta seção, conforme ilustrado na imagem abaixo.

  1. Ao clicar no botão, você poderá:

  • Escolher o novo status.

  • Adicione um comentário.

  • Decidir se deseja ou não notificar o cliente sobre a alteração.
