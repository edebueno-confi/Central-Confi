COMO ATUALIZAR OS DADOS DE INTEGRAÇÃO DO E-COMMERCE

Para atualizar os dados de integração, siga os passos abaixo:

  1. Acesse o painel administrativo da sua loja.

  2. No menu lateral, vá até CADASTRO e selecione a aba E-COMMERCE.

  3. Dentro dessa aba, você poderá cadastrar os dados necessários para integrar sua loja com a plataforma utilizada.
