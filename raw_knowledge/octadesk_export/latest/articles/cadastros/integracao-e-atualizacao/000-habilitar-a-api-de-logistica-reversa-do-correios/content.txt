Como habilitar a API de Logística Reversa do Correios?
 Para que a integração com os Correios funcione corretamente na Genius Returns, é necessário que seu contrato tenha a API de Logística Reversa (LR250) habilitada. Para verificar se a API está ativada, acesse a tela dos Correios e procure pela seguinte informação:

 O que acontece se a API não estiver habilitada?
 Caso a API de Logística Reversa (LR250) não esteja habilitada, não será possível integrar os dados do seu contrato com a Genius Returns. Nesse caso, você verá o seguinte erro:

 Se a sua API de Logística Reversa não estiver habilitada, basta enviar um e-mail para a sua agência dos Correios solicitando a liberação da API. A solicitação não tem custo algum.
 Modelo de e-mail para solicitação de habilitação:

 Assunto: Liberação LR250 - Solicitação
 Corpo do e-mail:
 Prezados,
 Solicito a liberação do componente de Logística Reversa - LR250 no meu contrato, com os seguintes dados:
 Razão Social: [Sua Razão Social]
 CNPJ: [Seu CNPJ]
 Contrato: [Número do contrato]
 Código Administrativo: [Código administrativo]
 Cartão Postal: [Número do Cartão Postal]
 Agradeço pela atenção e aguardo a confirmação da liberação.

 Após o envio da solicitação, a sua agência dos Correios deve processar o pedido e habilitar a API em breve.
